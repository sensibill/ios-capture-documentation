Changelog

Release 1.26.3

  • Xcode 12.5 compatible.

Release 1.26.1

New Features

  • A UIImage extension now provides a jpegDataForSensibillAPI() method, which converts a UIImage, returned by the Capture module, to JPEG Data that is compatible with Sensibill’s API. See Integration for more information.

Compatibility

  • Xcode 12.4 is now supported.
  • CaptureNavigationControllerDelegate can now be implemented by an Objective-C class

Customization

  • Resolved an issue where some buttons could not be seen on a light background

Accessibility updates

  • Resolved an issue where the Previous Page indicator was missing its accessibility hint.

Release 1.26.0

Accessibility updates

  • VoiceOver announcements for the auto-capture button have changed. Intead of “Auto-Capture on/off/disabled”, the statements have been clarified: “Auto-Capture is on/off/disabled”
  • VoiceOver announcements for the flash button have changed. Intead of “Flash on/off”, the statements have been clarified: “Flash is on/off”

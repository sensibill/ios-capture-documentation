Capture Colors

Colors and configs contained in the CaptureTheme class are ones that apply to specific UI elements in a specific section of the module. For example, the done button on the crop screen and capture preview screen are both Primary Buttons, so to make one differ visually from the other you would have change one of their corresponding colors/configs in the CaptureTheme module, as changes to primary buttons in the DefaultTheme will apply to them both. A CaptureTheme can be initialized with a DefaultTheme to provide default color mappings different from the set defaults.

Font Customization

The DefaultTheme object has a property called fonts that allows the integrator to override the three default font styles (Regular, SemiBold, and Bold) via the defaults property by passing a FontSet. The integrator also has more fine-grained control and can override the font styles used throughout the module. For a full list of customizable fonts, see the documentation for the FontManager class.

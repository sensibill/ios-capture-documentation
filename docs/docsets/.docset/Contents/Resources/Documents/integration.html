Integration

Overview

Sensibill Capture is a standalone framework or static library that allows for the scanning and cropping of receipts and documents. The module can scan and return up to three documents simultaneously to the integrator’s app.

Importing Capture Module

Swift

import SensibillCapture

Objective-C

@import SensibillCapture;

Note: CocoaPods must be used if you want to include Capture as a static module. In this case use the following import statement in swift.

import SensibillCaptureStatic

Implementing a Capture Delegate

Capture provides a CaptureNavigationControllerDelegate protocol to return an array of captured images back to the integrator’s app. The protocol can be implemented as an extension to an existing class, which instantiates capture (as shown below), or in a different class.

The CaptureNavigationControllerDelegate will return 1-3 photos captured by a user in an images array. If a user cancels the operation, or an error occurrs the images array will be empty.

extension YourClass: CaptureNavigationControllerDelegate {

     func captureNavigationController(_ controller: CaptureNavigationController, didCapture images: [UIImage]) {

        print("Received \(images.count) photos")
    }
}

Sending Images to the Sensibill API

One of the common use cases for the Capture module is sending the resulting images to Sensibill’s API for receipt processing. To ensure a successful processing of the receipt by Sensibill’s API, the images must comply with the size and quality requirements specified in Sensibill API documentation.

The Capture module provides a jpegDataForSensibillAPI() method in a UIImage extension, which converts images to a format optimal for the Sensibill API. The method can be used upon receival of the images in the didCapture method described above:

func captureNavigationController(_ controller: CaptureNavigationController, didCapture images: [UIImage]) {

    for image in images {

        let jpegData = image.jpegDataForSensibillAPI()
        // Submit jpegData to Sensibill API
    }
}

Presenting Capture

The module starts with the camera preview, and is presented modally. To initialize and present the module:

  1. Import SensibillCapture or SensibillCaptureStatic as explained above.
  2. Instantiate CaptureNavigationController
  3. Select a modalPresentationStyle for the CaptureNavigationController.
  4. Set a captureDelegate for CaptureNavigationController.
  5. Present the CaptureNavigationController.

In the following example, overFullScreen was selected as the modalPresentationStyle, and captureDelegate is set to the class itself, assuming extension YourClass: CaptureNavigationControllerDelegate was implemented:

import SensibillCapture
// or SensibillCaptureStatic if using the static library option

...

class YourClass {

...

    let captureNavigationController = CaptureNavigationController()
    captureNavigationController.modalPresentationStyle = .overFullScreen
    captureNavigationController.captureDelegate = self
    present(captureNavigationController, animated: true, completion: nil)

...

}

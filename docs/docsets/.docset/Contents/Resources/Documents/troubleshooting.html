Compilation error Building for the iOS Simulator, but the linked and embedded framework 'SensibillCapture.framework' was built for iOS + iOS Simulator. appears when running on simulator using Xcode 12.3

  • Update target’s Build Settings Validate Workspace settings to Yes under Build Options section.

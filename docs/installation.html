Installation

Overview

The Sensibill Capture module is available in a private repository on bitbucket. Please contact Sensibill to obtain access to the repository.

Once access is obtained, you can either use the SDK in your project via CocoaPods or by downloading and importing either the static or dynamic .framework file into your project.

Via CocoaPods

CocoaPods provides a quick way to download external dependencies for an iOS project. The Sensibill Capture Module can be downloaded and accessed via CocoaPods:

  1. If your project isn’t using CocoaPods already, follow the set up guide from the official CocoaPods website here

  2. Once your project is set up with CocoaPods, add the Sensibill dependency to your project’s Podfile in the following format: pod 'Sensibill', :git => 'ssh://git@bitbucket.org:sensibill/sensibill-ios-capture-release.git', :tag => 'vX.Y.Z', where vX.Y.Z represents to version of the module you want to use.

  3. In your terminal, navigate to the root directory of your application’s project folder.

  4. Run pod install to integrate Sensibill SDK to your project.

Direct Download

To download the Sensibill Capture module directly, contact our Delivery/Support team to get access to the portal to download the SensibillCapture.framework or the SensibillCaptureStatic.framework file.

  • Download and Import one of the .framework files into your project, and ensure that it is included in Embedded Binaries and Linked Frameworks and Libraries in your project settings

Using SensibillUI module in Swift

import StandaloneCapture

OR

import StandaloneCaptureStatic

Using SensibillUI module in Objective-C.

When the Sensibill Capture framework is used in an Objective-C Project, the ‘SensibillCapture’ framework can be imported by adding the following statement.

@import SensibillCapture;

OR

@import SenisbillCaptureStatic;
